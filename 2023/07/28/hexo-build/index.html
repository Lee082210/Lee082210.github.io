为hexo博客配置自己的私人域名

发表于2023-07-28|更新于2023-07-28
|字数总计:481|阅读时长:1分钟|阅读量:

1. 域名设置

本篇文章以阿里云域名为例，如果还没有域名的可以自行购买（一般的域名不会太贵），其他域名配置也是大同小异，大家百度即可。

1.在阿里云控制台中找到域名设置，找到域名行所属的 解析项 点击解析。

2.进入云解析NDS后，我们点击 添加记录 添加两条记录即可。一条记录为CNAME类型（如果需要将域名指向另一个域名，再由另一个域名提供 IP 地址，就需要添加 CNAME 记录。），另一条记录为A记录 主机记录设置为www（用来指定域名的 IPv4 地址（如 8.8.8.8），如果需要将域名指向一个 IP 地址，就需要添加 A 记录。）

3.接下来 修改CNAME记录 将他的记录值修改为你的hexo博客网域名username.github.io这里的username为你github的账户名(如下图)，A记录修改你username.github.io对应的IP地址，如果不知道的话 进入cmd命令窗，执行 ping username.github.io 即可查询对应的IP，将他修改到A记录的记录值里。（启动后DNS解析可能会久一点才生效）

2. hexo本地设置

1.进入你创建的hexo本地目录下，右键 Git Bash Here，之后 cd 到source 文件夹下 cd source，
执行 touch CNAME命令创建CNAME文件，之后在目录下使用记事本编辑CNAME 将你的域名写进并且保存后 退出。

2.完成以上步骤后 将修改的本地配置提交到github上，执行 hexo g -d

3.完成所有步骤后，就可以访问域名啦，快去试试吧。
